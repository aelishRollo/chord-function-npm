Chord Function API Guide

Welcome to the Chord Function API! This API allows you to determine the chord function based on the key center, chord root, and chord quality. This is an API written by a nerd, but whether you're a nerd or not I hope this is of use to you.

How to Use

To get the chord function for a G Major7 chord in the key of C, make the following request:

GET https://chord-function-calculator-03b3fc20902a.herokuapp.com/API?keyCenter=C&chordRoot=G&quality=Major7

Response

The response will be a JSON object containing the chord function:

{
    "chordFunction": "Vmaj7"
}

Query Structure

The API uses query parameters to receive input. Here's a breakdown of the structure:

  • The base URL is followed by /API?
  • Each parameter is in the format key=value
  • Parameters are separated by the ampersand symbol &

For example, in the URL:

https://chord-function-calculator-03b3fc20902a.herokuapp.com/API?keyCenter=C&chordRoot=G&quality=Major7
  • keyCenter=C is the first parameter
  • & separates the first and second parameters
  • chordRoot=G is the second parameter
  • & separates the second and third parameters
  • quality=Major7 is the third parameter

Query Parameters

  • keyCenter - The key center (e.g., C, G, F#)
  • chordRoot - The root of the chord (e.g., C, G, F#)
  • quality - The quality of the chord (e.g., Major, Major7, Minor)

Notes with Sharps

When using notes with sharps (e.g., C#, F#), you need to encode the '#' symbol in the URL. Use '%23' instead of '#'. For example:

  • C# becomes C%23
  • F# becomes F%23

Example request with a sharp note:

GET https://chord-function-calculator-03b3fc20902a.herokuapp.com/API?keyCenter=F%23&chordRoot=B&quality=Minor7

Testing the API

You can test the API using a tool like Postman or cURL.

Using cURL

curl -X GET "https://chord-function-calculator-03b3fc20902a.herokuapp.com/API?keyCenter=C&chordRoot=G&quality=Major7"

Using cURL with Sharp Notes

curl -X GET "https://chord-function-calculator-03b3fc20902a.herokuapp.com/API?keyCenter=F%23&chordRoot=B&quality=Minor7"
